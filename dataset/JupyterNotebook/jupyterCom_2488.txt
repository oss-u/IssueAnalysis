Launch the command prompt from Anaconda by clicking on “Open Terminal” (from any environment)\r\n2. Open \"jupyter_notebook_config.py\" file and search for “#c.NotebookApp.notebook_dir = ' ' ‘‘\r\n4. Remove the comment and add your desired path inside quotes with double slashes (\\\\). Suppose I want to point it to E drive, it would look like this: c.NotebookApp.notebook_dir = 'E:\\\\'\r\n5. Save it and done with Anaconda.\r\nYou can stop here, if you don't use any shortcut icons.\r\n\r\nUpdate the Anaconda icon:\r\n1. Open Jupyter Notebook shortcut location from start Menu\r\n2.
