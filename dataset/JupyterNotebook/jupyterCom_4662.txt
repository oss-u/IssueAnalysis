    "body": "Wanted to make a JetBrains keymap on Jupyter Notebook. Had to change how CodeMirror treats my keyboard shortcuts by using [custom.js](https://jupyter-notebook.readthedocs.io/en/latest/examples/Notebook/JavaScript%20Notebook%20Extensions.html#custom.js). \r\n\r\nThis link gives the list of CodeMirror commands : https://codemirror.net/doc/manual.html\r\n\r\nThe following lines of code work in order to :\r\n\r\n* Duplicate a line with<kbd>Ctrl</kbd> + <kbd>D</kbd> (thanks to #1816 )\r\n* Delete a line with<kbd>Ctrl</kbd> + <kbd>Y</kbd>\r\n\r\n```js\r\n/**\r\n*\r\n* Duplicate a current line in the Jupyter Notebook\r\n* Used only CodeMirror API - https://codemirror.net\r\n*\r\n**/\r\nCodeMirror.keyMap.pcDefault[\"Ctrl-D\"] = function(cm){\r\n\r\n    // get a position of a current cursor in a current cell\r\n    var current_cursor = cm.doc.getCursor();\r\n\r\n    // read a content from a line where is the current cursor\r\n    var line_content = cm.doc.getLine(current_cursor.line);\r\n\r\n    // go to the end the current line\r\n    CodeMirror.commands.goLineEnd(cm);\r\n\r\n    // make a break for a new line\r\n    CodeMirror.commands.newlineAndIndent(cm);\r\n\r\n    // filled a content of the new line content from line above it\r\n    cm.doc.replaceSelection(line_content);\r\n\r\n    // restore position cursor on the new line\r\n    cm.doc.setCursor(current_cursor.line + 1, current_cursor.ch);\r\n};\r\n/**\r\n*\r\n* Delete a current line in the Jupyter Notebook\r\n* Used only CodeMirror API - https://codemirror.net\r\n*\r\n**/\r\nCodeMirror.keyMap.pcDefault[\"Ctrl-Y\"] = function(cm){\r\n\r\n    // delete line\r\n    CodeMirror.commands.deleteLine(cm);\r\n};\r\n```",
