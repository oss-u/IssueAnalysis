    "body": "I manged to solve it. Here is what I did.\r\n\r\n1. Add the _remove_cell_ tag to the cell where you define a command or declare a math operator. Applying a tag it is possible to hide the cell using the tag editor built into the notebook. Suppose I have a cell used only to define the sign math operator.\r\n`$\\DeclareMathOperator{\\sign}{sign}$`\r\nAdding this cell makes it possible to use the operator sign in the notebook, but the command will appear in LaTeX outside the preamble, causing error. Therefore we need to remove it when converting the notebook.\r\n\r\n2. Create a template. Edit a file _temp.tplx_:\r\n\r\n```\r\n((*- extends 'article.tplx' -*))\r\n\r\n((* block author *))\r\n\\author{My Name}\r\n((* endblock author *))\r\n\r\n((* block title *))\r\n\\title{My Document Title}\r\n((* endblock title *))\r\n\r\n((* block definitions *))\r\n((( super() )))\r\n\\DeclareMathOperator{\\sign}{sign}\r\n((* endblock definitions *))\r\n```\r\n\r\n3. Use nbconvert and remove the tags marked with the tag _remove_cell_.\r\n\r\n```\r\njupyter nbconvert --to pdf --template temp notebook.ipynb --TagRemovePreprocessor.remove_cell_tags='{\"remove_cell\"}' \r\n```",
